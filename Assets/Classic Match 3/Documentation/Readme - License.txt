HOW TO USE

Game must be started from Menu scene.

This template has a modular structure. You can edit or easily replace gameplay files with no affection to menu,
sound, storage and other components.

Images and sounds that are used in this project is under CC0 license. You can keep them or replace with your own.

If you want to use it as iOS application just make a build and publish it with Xcode. Everything is already set.
Do not forget to change bundle identifier into Project Settings -> Player tab.
